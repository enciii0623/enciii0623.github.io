# 林恩慈个人主页

这是一个可以直接部署到 GitHub Pages 的静态个人主页。

## 文件结构

```text
index.html
assets/
  styles.css
```

## 部署方式

1. 在 GitHub 新建仓库 `enciii0623.github.io`。
2. 将本目录下的所有文件上传到仓库根目录。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub 生成访问链接。

如果仓库名是 `enciii0623.github.io`，访问地址通常是：

```text
https://enciii0623.github.io/
```

如果仓库名是普通项目仓库，例如 `personal-homepage`，访问地址通常是：

```text
https://你的GitHub用户名.github.io/personal-homepage/
```
